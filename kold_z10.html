Temat: Budowa selektora

  1. Wymień oraz opisz dwie części selektora

    Selektor wskazuje element(y) HTML, które mają zostać sformatowane, np. p, .menu, #naglowek.

    Deklaracja znajduje się w klamrach i zawiera cechy oraz wartości, np. color: red;.

  2. Znaczenie nawiasów klamrowych { ... }

    Nawiasy klamrowe otwierają i zamykają blok deklaracji stylu przypisany do danego selektora.

  3. Opisz cechy

    Cechy (właściwości) to nazwy parametrów wyglądu, np. color, font-size, background-color, margin.

  4. Opisz wartości

    Wartości określają, jak dana cecha ma wyglądać, np. blue, 16px, bold, #FF0000.

  5. Kiedy stosujemy średnik (;) a kiedy nie jest konieczny?

    Średnik stawiamy po każdej deklaracji cechy. W ostatniej deklaracji bloku bywa opcjonalny, ale zaleca się go zawsze pisać.

Przykład pełnej reguły:

p {
    color: navy;
    font-size: 18px;
}
