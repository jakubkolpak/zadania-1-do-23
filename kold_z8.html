1. Co to jest wewnętrzny arkusz stylów?

    Wewnętrzny arkusz stylów to zestaw reguł CSS umieszczonych w sekcji head strony, wewnątrz znacznika style. Działa tylko w obrębie jednego dokumentu HTML.

  2. Gdzie umieszczamy wewnętrzny arkusz stylów i jaka jest składnia?

    Wewnętrzny arkusz stylów umieszczamy między znacznikami <head> i </head>: <style> selektor { cecha: wartość; } </style>.

  3. Jakie są zalety i wady wewnętrznego arkusza stylów?

    Zaletą jest łatwe zarządzanie wyglądem jednej strony i brak dodatkowego pliku CSS. Wadą jest brak współdzielenia stylów między wieloma stronami oraz większy kod HTML.

  4. Różnica między stylem lokalnym, wewnętrznym i zewnętrznym.

    Styl lokalny dotyczy pojedynczego elementu, wewnętrzny dotyczy jednej strony, a zewnętrzny (plik .css) może obsługiwać wiele stron jednocześnie.

  5. Przykład reguły CSS wewnątrz style.

    Przykład: p { color: blue; font-size: 18px; } — oznacza, że wszystkie akapity p na stronie będą miały niebieski kolor i rozmiar 18px.
