Użycie ID

Pojęcie Znaczenie pojęcia uzupełnia uczeń
Co znaczy skrót ID? ID = Identifier (identyfikator).
Jaki jest cel stosowania ID? Jednoznaczne wskazanie wybranego elementu na stronie i przypisanie mu stylu.
Jakie jest znaczenie znaku # W CSS znak # oznacza selektor identyfikatora.
Jak następuje wywołanie ID (wraz z wytłumaczeniem)? W CSS: #nazwa_id { ... } oraz w HTML: id="nazwa_id".
Uwagi na temat stosowania ID (trzy) konieczne wyliczenia.
  1. ID powinno być unikalne na stronie.
  2. ID ma wyższy priorytet niż selektor znacznika.
  3. Do wielokrotnego użycia lepiej stosować class niż id.

Przykład

Selektor ID: #pochyly, użycie w HTML: id="pochyly".
