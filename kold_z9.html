Temat: Opis stylu zewnętrznego

  1. Składnia teoretyczna arkusza CSS

    selektor {
    cecha1: wartość1;
    cecha2: wartość2;
}

    Selektor wskazuje element HTML, a w nawiasach klamrowych umieszczamy deklaracje stylu.

  2. Gdzie wstawiany jest styl zewnętrzny i ile razy może wystąpić?

    Styl zewnętrzny dołączamy w sekcji <head> dokumentu przez znacznik <link>. W jednym dokumencie można dołączyć wiele arkuszy CSS.

  3. Najprostsza forma wstawiania stylu zewnętrznego

    <link href="nazwa_pliku.css" rel="stylesheet" type="text/CSS">

    Taki zapis umieszczamy wewnątrz sekcji <head>.

  4. Czy można dołączyć kilka arkuszy CSS i jaka zasada obowiązuje?

    Tak. Można podłączyć wiele arkuszy. Przeglądarka czyta je od góry do dołu, a przy konflikcie zwykle wygrywa reguła późniejsza (o ile specyficzność jest taka sama).

  5. Co przy konflikcie tych samych stylów?

    Decyduje kaskadowość CSS: ważność (!important), specyficzność selektora, a następnie kolejność deklaracji.

  6. Jak definiuje się style w różnych arkuszach dla tego samego elementu?

    Każdy arkusz może zawierać własne reguły. Dla tego samego selektora obowiązuje ta reguła, która ma wyższy priorytet według zasad kaskady.

  7. Jak tworzy się i edytuje pliki CSS w edytorze tekstowym?

    Tworzymy zwykły plik z rozszerzeniem .css, zapisujemy reguły CSS i edytujemy go tak jak plik tekstowy.

  8. Jak tworzy się i edytuje pliki CSS w specjalnym edytorze?

    W edytorach CSS/IDE można korzystać z podpowiedzi składni, kolorowania kodu i automatycznego formatowania.

  9. Jakie rozszerzenie posiada plik arkusza styli?

    Plik arkusza styli ma rozszerzenie .css.

  10. Dwie zalety zewnętrznego arkusza styli

    1. Jedna zmiana w pliku CSS może zmienić wygląd wielu stron jednocześnie.
    2. Kod HTML jest czytelniejszy, bo oddzielamy treść od wyglądu.
